About noUiSlider

noUiSlider is a little jQuery plugin that makes really cool (double) range sliders. Every slider can have two handles to select a range, a fixed minimum or maximum can be set to select a limit, or two handles can be used, to simply pick some points.

Using noUiSlider

Initiating a basic slider:

  $('#sliderExample').noUiSlider('init');

Options

Name Used in functions Parameter type Possible parameter values Description
knobs init [INT] 1, 2 Specifies the number of knobs.
connect init [MIXED] "upper", "lower", false, true Whether to connect the middle bar to the knobs.
scale init, move, value [ARRAY] An array containing two values. Both values can be positive or negative, but the first value must be lower than the second. The values represented by the slider knobs.
start init [ARRAY] An array containing two values, relative to the values given in {scale}. The starting positions for the knobs, mapped to {scale}.
to move [INT] Any value, relative to {scale}. The value will be corrected to fit within the edges of the slider. The upper knob will not be place below and lower knob, and vice versa. The position to move a knob to.
knob move [MIXED] 0,1,"lower","upper" The knob to move.
change init [FUNCTION] A 
function(){...}
declaration, or a reference to a function object: 
var doSomething = function(){...}
 The function to be called on every change.
end init [FUNCTION] Same as above ^ The function when a knob is no longer being changed.
step init [INT] Any value smaller than the difference between {start[0]} and {start[1]}. If you pick a value too big or too small relative to your {scale} setting, this option might not make sense. Whether, and at what intervals, the slider should snap to a new position. Adheres to {scale}
save move, value [BOOLEAN] true, false Whether a scale give to a function should become the default for the slider it is called on.
