Overview

An API to deliver filtered records from a saved report.

Summary

This endpoint will return a list of records from a saved report filtered by an expression. Currently filters are used for chart drill through only.

Request Arguments

Name Type Description Required
filter String The filter expression. Filter can be specified as query parameters. Example: filter[0][industry][$in][]=Engineering. False

Request Example

GET /rest/v10/Reports/:id/records

Response Arguments

Name Type Description
records Array An array of results containing matched records.
