Get interactions for current contact record

Summary:

This endpoint is used to return the json Data for the Contact interactions widget.

Query Parameters:

Param Description Optional
view Name of view which uses with limit. Allowed values: emailsRecv, emailsSent, meetings, calls true
limit A maximum number of records to return for selected view true
filter Special filter for filtering data. Value "favorites" - only fetch favorite records. If uses numerical value then this value uses for fetching records by last days true
list Filter for filtering records with assigned user criteria. Allowed values: my, all. "my" - records which assigned to current user. "all" - all records true

Input Example:

{ "list":"all", "filter": 30, "limit": 5, "view": "emailsSent" }
