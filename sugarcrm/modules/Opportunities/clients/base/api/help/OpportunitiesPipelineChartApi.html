Get the current users personal or team opportunity pipeline

Summary:

This endpoint is used to pipeline chart data for the Pipeline widget

Query Parameters:

Param Description Optional
timeperiod_id Which Timeperiod to View, defaults to the current timeperiod if not defined true
Type What type of pipeline to view, valid options are self which will just show your pipeline and team which will show you and everyone under the current user, defaults to self if not defined true

Valid Urls:

  • /rest/v10/Opportunities/chart/pipeline/
  • /rest/v10/Opportunities/chart/pipeline/:timeperiod_id
  • /rest/v10/Opportunities/chart/pipeline/:timeperiod_id/:type
