Overview

Creates a relationship to a pre-existing email. See RelateRecordApi for details.

Links

Link Description
accounts_from Existing accounts may be linked. An email may only have one sender. Linking an account replaces the existing sender, if any. The sender cannot be changed after the email is archived. An email_address or email_address_id can be provided along with the record to specify the email address used for the sender, in the event that it was not the account's primary email address.
