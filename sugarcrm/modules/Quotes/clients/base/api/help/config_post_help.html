Overview

Quote Config

Summary

This endpoint allows customizations to be made to the Quoted Line Items section of the Quote record.

Request Arguments

Name Type Description Required
worksheet_columns array A viewdef definition of the columns you want to display in the QLI section. This needs to include all of the keys that you would include in a filesystem based viewdef (see the example below) True
worksheet_column_related_fields array Array of the related fields needed to be returned so that your list of fields can be properly displayed. True
