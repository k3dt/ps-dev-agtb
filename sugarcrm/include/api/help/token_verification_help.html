Overview

Verifies and saves server token.

Request Arguments

Name Type Description Required
id String The ID of server who needed to verify. The "socket" or "trigger" values available now. True
original String Original token was sent to server True
verified String New token needed to store and to use in next requests True

Request for token verification

{
    "id":"socket",
    "original":"964db99c9884ea5fa77bc60fcb6e0d75",
    "verified":"c96eb75d-c2b0-4f71-90de-9a01457880eb"
}

{
    "id":"trigger",
    "original":"964db99c9884ea5fa77bc60fcb6e0d75",
    "verified":"c96eb75d-c2b0-4f71-90de-9a01457880eb"
}

Response

There is no response body.
