Export records

Summary:

This endpoint returns a set of records in CSV format. It also returns several http headers to indicate content properties.

Query Parameters:

ParamDescriptionOptional
uidOne or an array of bean ids.Optional
filterThe filter expression. Filter expressions can be found in filter api document.Optional
sampleBoolean to indicate whether to download sample data.Optional

Input Example:


This would filter the records in the module and find records that had the name "Avery Software Co" and export the data in csv format.
{ "filter":[{"name":"Avery Software Co"}] }
This would export a single record with the given id.
{ "uid":"9279a9cb-48cd-a1fb-beae-50c7bd9247b5" }
This would export records with the array of given ids.
{ "uid":["9279a9cb-48cd-a1fb-beae-50c7bd9247b5","c85fe675-0c09-ce23-9ec8-50c7bdee969c"] }
This would export the sample records with some description.
{ "sample":true }
