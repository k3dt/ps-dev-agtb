Update/delete one or more records at the same time

Summary:

This endpoint updates or deletes one or multiple records at the same time.

Query Parameters:

ParamDescriptionOptional
massupdate_paramsMass update parameters that include the information below
modulemodule name
uidcomma separated ids
[field name]field to be modifiedoptional
team_nameteam arrayoptional
team_name_typereplace or add teamsrequred when team_name is used
entiretrue for all recordsoptional
current_searchcurrent search arrayoptional
deletetrue to delete recordsoptional, default is false

Input Example:

PUT parameters to update do_not_call field for contacts, given ids: "massupdate_params": array( "module":"Contacts", "uid":array("b57a80d7-be94-8f16-f875-5022a90e05fc,c4f0ac35-b23b-5ccb-e974-5022a90c08da"), "do_not_call":1, )
PUT parameters to replace team IDs and set primary team id for contacts, given ids: "massupdate_params": array( "module":"Contacts", "uid":array("b57a80d7-be94-8f16-f875-5022a90e05fc,c4f0ac35-b23b-5ccb-e974-5022a90c08da"), "team_name":array(0 => array('id' => $team1->id, 'primary' => true), 1 => array('id' => $team2->id, 'primary' => false),), "team_name_type":"replace", )
PUT parameters to update do_not_call field for all contacts: "massupdate_params": array( "module":"Contacts", "entire":true, "do_not_call":1, )
PUT parameters to update do_not_call field for all contacts, with basic search ('name'='airline'): "massupdate_params": array( "module":"Contacts", "entire":true, "do_not_call":1, "current_search":array("search_type"=>"basic", "name"=>"airline"), )
PUT parameters to update do_not_call field for all contacts, with advanced search ('first_name'='airline' and 'assigned_user_id'=user1 or user2): "massupdate_params": array( "module":"Contacts", "entire":true, "do_not_call":1, "current_search":array("search_type"=>"advanced", "first_name"=>"airline", "assigned_user_id"=>array(0=>id1,1=>id2)), )
DELETE parameters to delete contacts, given ids: "massupdate_params": array( "module":"Contacts", "delete":true "uid":array("b57a80d7-be94-8f16-f875-5022a90e05fc,c4f0ac35-b23b-5ccb-e974-5022a90c08da"), )
DELETE parameters to delete all contacts: "massupdate_params": array( "module":"Contacts", "delete":true "entire":true, )
DELETE parameters to delete all searched contacts, using basic search ('name'='airline'): "massupdate_params": array( "module":"Contacts", "delete":true "entire":true, "current_search":array("search_type"=>"basic", "name"=>"airline"), )
DELETE parameters to delete all searched contacts, using advanced search ('name'='airline' and 'assigned_user_id'=user1 or user2): "massupdate_params": array( "module":"Contacts", "delete":true "entire":true, "current_search":array("search_type"=>"advanced", "first_name"=>"airline", "assigned_user_id"=>array(0=>id1,1=>id2)), )

Output Example:

ok
