Download a vCard

Summary:

This endpoint downloads a vCard.

Query Parameters:

ParamDescriptionOptional
idUnique idYes, default it Contacts
modulemodule name

Input Example:

GET parameters to download a vCard, given a contact id: "module":"Contacts", "id":"b57a80d7-be94-8f16-f875-5022a90e05fc"

Output Example:

BEGIN:VCARD N;CHARSET=utf-8:SugarContactLast;SugarContactFirst1456886181;; FN;CHARSET=utf-8: SugarContactFirst1456886181 SugarContactLast BDAY: TEL;WORK;FAX: TEL;HOME: TEL;CELL: TEL;WORK: EMAIL;INTERNET;CHARSET=utf-8:contact@1456886181sugar.com ADR;WORK;CHARSET=utf-8:;;;;;; ORG;CHARSET=utf-8:; TITLE: END:VCARD
