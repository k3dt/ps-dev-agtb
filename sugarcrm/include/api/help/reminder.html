Overview

Reminds the user subscribed to the call or the meeting.

Request Arguments

Name Type Description Required
class String The class of the bean (Call or Meeting). True
beanId String The id of Call or Meeting. True
userId String The id of User's bean needed to remind. True

Request for Call/Meeting reminder

{
   "class":"Call",
   "beanId":"c96eb75d-c2b0-4f71-90de-9a01457880eb",
   "userId":"dab0219f-2452-4515-0d8d-562e59937c93"
}

Response

There is no response body.
