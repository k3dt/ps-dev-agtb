Project Template

Use this page to create a project template. After you create a template, you can save it as a new project whenever necessary.

To create a project template, enter the following information:

  • Name. Enter a name for the project template.
  • Status. From the drop-down list, select the template's status.
  • Start Date. Enter the template's start date; to select from the calendar, click the calendar icon and select the date.
  • End Date. Enter the template's end date; to select from the calendar, click the calendar icon and select the date.
  • Team. Enter the name of the team that is allowed to view the template; to select from the Team list, click Select and click the team name.
  • Priority. From the drop-down list, select the template's priority.
  • Assigned to. Enter the name of the user who has ownership of the template. By default it is assigned to you.
  • Description. Enter a brief description of the template.
To create the template, click Save; click Cancel to exit the page without creating the template.
