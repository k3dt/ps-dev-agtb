Forums

To create a new forum, enter the following information:

  • Title. Enter a name for the forum.
  • Team. Click Select and from the Team list, select the team that will have permissions to view this forum.
  • Description. Enter a brief description of the forum.
  • Create Forum Topic. Select an existing topic for the forum from the drop-down list or click the link to create a new topic.

Click Save to create the forum; click Cancel to exit the page without creating the forum.
