Knowledge Base

Manage existing tags and articles from this page. Tags, which are used to group articles, can be either root tags or sub-tags. Sugar Suite provides a FAQs root tag. The number within parentheses next to each tag name indicates the number of associated articles.

  • To view a list of articles associated with the article, select the tag from the Tags column. A list of articles display in the right panel.
  • To view the contents of an article, point the cursor at its title; to view the article's detail page, click its title.
  • To create a tag, do the following:
    1. Select Create New Tag from the Admin Actions drop-down list.
    2. To create a new root tag, select Tags, enter the tag name in the text field above; to create a sub-tag select an existing tag, enter the tag name in the text field
    3. Click Save.
      A root tag is listed under Tags and a sub-tag is listed under the parent tag.
  • To rename a tag, do the following:
    1. Select Rename Tag from the Admin Actions drop-down list.
    2. Select the tag from the Tags list and enter the new name in the text field above.
    3. Click Save.
  • To delete a tag, do the following:
    1. Move or delete all articles that the tag contains.
    2. Select Delete Tag from the Admin Actions drop-down list and then select the tag that you want to delete
    3. Click Delete.
  • To move articles, do the following:
    1. Select Move Selected Articles from the Admin Actions drop-down list.
    2. Select the tag from the Tags list to view the articles that it contains.
    3. Select the articles from the right panel and click Select Tag.
    4. From the Tags window, select a new tag for the articles and click Move.
  • To delete articles, do the following:
    1. Select Delete Selected Articles from the Admin Actions drop-down list.
    2. Select the tag that contains the articles.
    3. Click Delete.
  • To create new tags for an article, do the following:
    1. Select the tag that currently contains the article.
    2. Select Apply Tags to Articles from the Admin Actions drop-down list.
    3. Select the article from the right panel and click Select Tag.
    4. Select one or more tags from the Tags window and click Apply Tag.
