Portal

Use the My Sites module to add links to Web sites and to Web-based applications such as forums or the Web interface to your organization's email system.

The My Sites Home page displays the following information:

  • A list of existing sites along with related information such as the Website and site type.
    • To view the details of a site, click its name in the Sites List.
    • To delete multiple sites, select them from the list, and click Delete.
  • A Shortcuts menu that displays the following options:
    • Add Site. Click this option to create a new portal.
    • List Sites. Click this option to navigate back to the Portal Home page from a site's detail page.

To create a site, enter the following information.

  • Name. Specify a name for the site.
  • Website. Enter the URL of the website.
  • Visible. To make it visible to users, select the Visible checkbox.
  • Placement. From the drop-down list, select where you want to display the link to the new site. You can select the Tab menu, the Shortcuts menu, or both.
  • Type. From the Type drop-down list, select the site type. Select Personal if you do not want to share the site other users; select Global if you want to make it visible to all users.

Click Save to create the site.; click Cancel to exit the page without creating it.

Sites that you create are listed in the Shortcuts menu.
